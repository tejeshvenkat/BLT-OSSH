# Contributing to BLT-OSSH

Thank you for your interest in contributing to BLT-OSSH (Open-Source Sorting Hat)!

## How to Contribute

### Submitting a Pull Request

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

### Contribution Areas

- **Project recommendations** — Add or improve project suggestions in the recommendation engine
- **Matching algorithm** — Improve how OSSH matches contributors to projects
- **Documentation** — Update README, add comments, or improve guides
- **UI/UX** — Enhance the interface, accessibility, or user experience

## Code Style

- Use consistent formatting (the project uses HTML, Tailwind CSS, and vanilla JavaScript)
- Keep changes focused — one PR per feature or fix
- Avoid overlapping file changes across multiple PRs to prevent merge conflicts

## Issues

- Report bugs or suggest features via [GitHub Issues](https://github.com/OWASP-BLT/BLT-OSSH/issues)
- Use the issue templates when available

## Questions?

Reach out via the [OWASP BLT discussions](https://github.com/OWASP-BLT/BLT/discussions) or open an issue.
